Compare sources

Pick any two readable sources and read them side-by-side. Each pane has its own search for verses, keywords, or phrases — type a verse like 9:5 or Quran 9:5, a hadith like 3731 or Bukhari 3731, a Bible ref like Genesis 15:6, or any word like crucifixion or jihad.

Every reader page opens inside the compare iframes in embed mode, so only the actual text shows — nav and footer are hidden.
