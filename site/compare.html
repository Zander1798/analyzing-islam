Compare sources

Pick any two readable sources and read them side-by-side. Each pane has its own search for verses, keywords, or phrases — type s9v5 (for Qur'an 9:5), genesis-15-6, sanhedrin-4-5, or any word like crucifixion or jihad.

Every reader page opens inside the compare iframes in embed mode, so only the actual text shows — nav and footer are hidden.
